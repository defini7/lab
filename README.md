## How to work with ScreenToOpenGL
Before 1 argument let's check declairing of `HWND hwnd`
1. argument - `hwnd`
2. argument - `LOWORD(lParam)`
3. argument - `HIWORD(lParam)`

before 4 and 5 arguments we need to declare `POINTFLOAT pf;`

4. argument - `pf.x`
5. argument - `pf.y`
